# SaaSify — Next.js SaaS Starter Template

A clean, modern, fully responsive SaaS starter template.

## Features
- Next.js 14 with App Router
- Tailwind CSS + shadcn/ui components
- Full dark mode support
- Responsive dashboard with sidebar
- Revenue chart with recharts
- Login / Register pages
- Settings page with tabs
- Landing page (Hero, Features, Pricing, FAQ, Footer)

## Tech Stack
Next.js 14, TypeScript, Tailwind CSS, shadcn/ui, Recharts, next-themes, Lucide React

## Getting Started
npm install
npm run dev

Open http://localhost:3000

## Pages
- / → Landing page
- /login → Login
- /register → Register
- /dashboard → Dashboard home
- /settings → Settings
